Types

Eachable1

Eachable which each() calls the callback with one argument

Parent types

  • Eachable - Parent type of all types that have each() method

Direct children types

  • Arr - Array - list of items accessed by zero-based index
  • ArrLike - Parent type for user-defined types with array-like behaviour. Use in cases when you would like to inherit from built-in Arr. Inheriting from built-ins is not possible for now.
  • Int - Integer type. On 64 bit platforms it's a 61 bit signed integer
  • Range
  • Str - String type
  • Table - Manipulates tabular data

Combine items to a single value using the supplied binary function First f is applied to start and the first element of something then on each step f is applied to previous result and next element of something.
