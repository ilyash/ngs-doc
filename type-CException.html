Types

CException

Base type for exceptions arising from errors returned by calling C functions

Parent types

  • Error - Represents an error. Usually more specific error types are used.

Direct children types

  • DirFail
  • FileIOFail
  • KillFail - Exception representing a failure to kill() a process
  • PipeCreateFail - Represents failure of creating a Pipe when calling PIPE(2)
  • StatFail - Represents a failure in stat() system call.
  • TtyCheckFail

Methods

init(e:CException, message:Str)
CException constructor. In addition to storing message attribute, adds errno and errno_name attributes.
