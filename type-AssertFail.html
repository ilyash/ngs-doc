Types

AssertFail

Represents failed asserttion, thrown by assert(). Should be used to catch programming errors.

Parent types

  • Exception - Represents exceptional situaution. All thrown things shouhld inherit Exception.

Example

F box_area(x:Int, y:Int) {
	assert(x > 0)
	assert(y > 0)
	x * y
}
echo(box_area(5, 10))
# Output: 50
echo(box_area(-5, 10))
# ... Exception of type AssertFail ...
