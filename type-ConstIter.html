Types

ConstIter

Iterator that returns given value forever.

Parent types

  • Iter - Iterator. Parent type for all specific iterator types.

Example

{ Iter }
i = ConstIter(10)
echo(i.next())
echo(i.next())
# Output:
#   10
#   10
