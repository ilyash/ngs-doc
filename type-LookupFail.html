Types

LookupFail

Represents an error of accessing non-existent element of a collection.

Parent types

  • Error - Represents an error. Usually more specific error types are used.

Direct children types

  • AttrNotFound - Represents an error of reading non-existent attribute of an object.
  • GlobalNotFound - Represents an error of accessing undefined global variable.
  • IndexNotFound - Represents an error of out-of-bounds array index.
  • KeyNotFound - Represents an error of accessing non-existent key in a hash.
