Convert "Tags" array in the given AWS resource (typically returned by AWS CLI) into Hash.
Source: stdlib.ngs:3090

Example

t = {"Tags": [{"Key": "k", "Value": "v"},{"Key": "k2", "Value": "v2"}]}  # Not very convenient, is it?
t.stdlib_aws_straighten_tags()
t  # {Tags={k=v, k2=v2}}
