Types

Sfx

Suffix subsequence of a sequence

Parent types

  • SubSeq

Direct children types

  • MaybeSfx - Suffix that might or might not be present for ~ matching to succeed
  • MustSfx - Suffix that must be present for ~ matching to succeed

Methods

-(s:Str, sfx:Sfx)
Return string without the suffix. Throws MatchFail if pfx is MustSfx but s does not end with it.
~(s:Str, sfx:Sfx)
Check whether s ends with pfx.
