Types

Iter

Iterator. Parent type for all specific iterator types.

Direct children types

  • ArrIter - Arr (array) iterator.
  • ConstIter - Iterator that returns given value forever.
  • HashIter - EXPERIMENTAL Hash iterator.
  • RangeIter - NOT IMPLEMENTED YET. Iterator that returns values in the given range

Methods

drop(i:Iter, n:Int)
Fetch and drop next n values from iterator i.
take(i:Iter, n:Int)
Fetch and convert to array next n values from iterator i.
