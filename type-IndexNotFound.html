Types

IndexNotFound

Represents an error of out-of-bounds array index.

Parent types

  • LookupFail - Represents an error of accessing non-existent element of a collection.

Direct children types

  • EmptyArrayFail - Represents an error of using an empty array for an operation that requires at least one element in the array.

Example

a = [10,20,30]
echo(a[100])
# ... Exception of type IndexNotFound ...
