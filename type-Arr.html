Types

Arr

Array - list of items accessed by zero-based index

Parent types

  • Eachable1 - Eachable which each() calls the callback with one argument

Example

x = ["first", "second", "third", "fourth"]

echo(x)
# Output:
#   ['first','second','third','fourth']

echo(x.len())
# Output:
#   4

echo(x[1])
# Output:
#   second

echo(x[10])
# ... Exception of type IndexNotFound occured ...
